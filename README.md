# Chokri Ghiloufi Football Portfolio

Static website files ready for GitHub + Vercel.

## Files
- `index.html` - main website content
- `style.css` - design and responsive layout
- `script.js` - mobile menu
- `assets/` - images and optional video assets

## Publish with GitHub + Vercel
1. Create a new GitHub repository.
2. Upload everything inside this folder, not the ZIP itself.
3. Open Vercel.
4. Click Add New → Project.
5. Import the GitHub repository.
6. Click Deploy.

## Replace photos later
Put your new photo in `assets/`, then edit the image path in `index.html`.
Example:
`assets/profile.jpg`
